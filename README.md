# Biophysics & Biomedical Engineering Lab Website

Static HTML website for the IIT Indore Biophysics & Biomedical Engineering Lab.

## Files

- `index.html` - complete website and admin panel

Edits made from the admin panel are saved in the browser with `localStorage`, so they remain after refresh on the same browser.
Firebase Firestore is also connected, so admin edits can be shared with all visitors after login.

Current admin features:

- Mobile-friendly public site and admin panel
- Gallery photo slideshow with editable slide timing
- Announcement bar speed, text color, and background color controls
- Hero background color/image and heading/subheading color controls
- Rich text editing for PI biography
- Refresh-safe saving for content, theme, users, media, gallery, announcements, and page names
- Team page grouped by category with editable category order
- Fullscreen image/video preview on tap
- Contact page with IIT Indore map and Prof. Sharad Gupta contact details
- Footer quick links with left/right logo controls

## GitHub Pages

1. Push this repository to GitHub.
2. Open repository `Settings`.
3. Go to `Pages`.
4. Select branch `main` and folder `/root`.
5. Save and open the generated GitHub Pages URL.
